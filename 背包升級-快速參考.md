# ⚡ 背包鐵砧升級系統 - 快速參考

## 🎒 背包等級

### 🟤 基礎背包 (9格)
- **獲取**: 工作台合成
- **配方**: 箱子 + 8皮革

### 🟦 標準背包 (27格)
- **獲取**: 鐵砧台升級
- **配方**: 箱子 + 基礎背包 + 鐵錠

### ⚫ 進階背包 (54格)
- **獲取**: 鐵砧台升級
- **配方**: 箱子 + 標準背包 + 鑽石

### 🟣 獄髓背包 (81格)
- **獲取**: 鐵砧台升級
- **配方**: 獄髓模板 + 進階背包 + 獄髓錠

---

## 🔧 升級步驟

### 步驟1: 製作基礎背包
```
工作台：
[皮革] [皮革] [皮革]
[皮革] [箱子] [皮革]
[皮革] [皮革] [皮革]
```

### 步驟2: 升級到標準
```
鐵砧台：
左上: 箱子
中間: 基礎背包
右上: 鐵錠
```

### 步驟3: 升級到進階
```
鐵砧台：
左上: 箱子
中間: 標準背包
右上: 鑽石
```

### 步驟4: 升級到獄髓
```
鐵砧台：
左上: 獄髓升級模板
中間: 進階背包
右上: 獄髓錠
```

## 🚀 快速測試

1. **構建**: `.\build-anvil-upgrades.bat`
2. **啟動**: `.\gradlew runClient`
3. **測試**: 工作台製作基礎背包，然後用鐵砧台升級

---

## 💡 提示

- 只有基礎背包可以用工作台合成
- 其他背包都需要鐵砧台升級
- 獄髓背包防火，不會被燒毀
