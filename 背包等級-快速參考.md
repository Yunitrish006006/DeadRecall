# ⚡ 背包等級系統 - 快速參考

## 🎒 三種等級背包

### 🟤 基礎背包
- **容量**: 9格 (1排)
- **合成**: 箱子外圍8個皮革

### 🟦 標準背包
- **容量**: 27格 (3排)
- **合成**: 基礎背包外圍8個皮革

### ⚫ 進階背包
- **容量**: 54格 (6排)
- **合成**: 標準背包外圍8個鐵錠

---

## 📋 合成配方

### 基礎背包
```
[皮革] [皮革] [皮革]
[皮革] [箱子] [皮革]
[皮革] [皮革] [皮革]
= 基礎背包 (9格)
```

### 標準背包
```
[皮革] [皮革] [皮革]
[皮革] [基礎] [皮革]
[皮革] [皮革] [皮革]
= 標準背包 (27格)
```

### 進階背包
```
[鐵錠] [鐵錠] [鐵錠]
[鐵錠] [標準] [鐵錠]
[鐵錠] [鐵錠] [鐵錠]
= 進階背包 (54格)
```

---

## 🚀 快速測試

1. **構建**: `.\gradlew build`
2. **啟動**: `.\gradlew runClient`
3. **獲取**: 創造模式 → 工具分類 → 三種背包
4. **測試**: 右鍵打開，驗證容量

---

## 💡 升級路徑

```
箱子 + 皮革
    ↓
基礎背包 (9格)
    ↓ + 皮革
標準背包 (27格)
    ↓ + 鐵錠
進階背包 (54格)
```
