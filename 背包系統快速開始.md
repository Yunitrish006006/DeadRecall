# 背包系統 - 快速開始指南

## 🎒 背包功能已成功實現！

### 功能摘要
- ✅ 註冊了背包物品
- ✅ 使用 Minecraft 原版袋子 (Bundle) 圖片
- ✅ 可以右鍵開啟背包UI
- ✅ 27個物品槽（3行×9列）
- ✅ 物品會自動儲存
- ✅ 防止背包套娃

---

## 🚀 立即測試

### 1. 啟動遊戲
```powershell
.\gradlew runClient
```

### 2. 獲取背包
在創造模式中：
1. 按 **E** 打開物品欄
2. 點擊 **🔨 工具** 分類標籤
3. 找到 **背包** 物品（使用袋子圖示）
4. 拖曳到快捷欄

### 3. 使用背包
1. 選中背包（在快捷欄）
2. **右鍵點擊** 開啟背包
3. 放入物品測試
4. 關閉後再次開啟，驗證物品保存

## 🔧 技術細節

### 使用的 API
- **物品註冊**: Fabric API - Item Registration
- **儲存系統**: Minecraft 1.21.1 Data Components API
- **介面系統**: ScreenHandler & HandledScreen
- **客戶端**: Fabric Client API

### 特色實現
1. **Data Components 儲存**: 使用 `DataComponentTypes.CONTAINER` 儲存物品，取代舊的 NBT 系統
2. **防止套娃**: 背包內不允許放置背包物品
3. **UI 繼承**: 繼承 `HandledScreen` 使用原版材質
4. **快速移動**: 支援 Shift+左鍵快速移動物品

## ❓ 疑難排解

### Q: 找不到背包物品？
A: 確認在創造模式物品欄的「工具」分類中，或使用 `/give` 指令

### Q: 開啟背包沒反應？
A: 確認是右鍵點擊，且不是在潛行狀態

### Q: 物品沒有保存？
A: 這可能是 bug，請檢查伺服器日誌，Data Components 應該會自動保存

### Q: 可以在生存模式使用嗎？
A: 可以！但需要先添加合成配方，或使用指令獲取
