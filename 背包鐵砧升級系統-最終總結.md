## 🔧 升級配方詳解

### 鐵砧台操作步驟
1. 打開鐵砧台
2. 左上角：放入**模板**物品
3. 中間：放入**基礎**物品（前一級背包）
4. 右上角：放入**附加**物品
5. 獲得升級後的背包

### 第一次升級：基礎 → 標準
```
模板：箱子
基礎：基礎背包
附加：鐵錠
結果：標準背包 (27格)
```

### 第二次升級：標準 → 進階
```
模板：箱子
基礎：標準背包
附加：鑽石
結果：進階背包 (54格)
```

### 第三次升級：進階 → 獄髓
```
模板：獄髓升級模板
基礎：進階背包
附加：獄隨錠
結果：獄髓背包 (81格，防火)
```

```

└─ 深入礦井挖鑽石

└─ 鐵砧台：箱子 + 標準背包 + 鑽石 → ⚫ 進階背包 (54格)
```

### 階段4：終極遊戲
```
探索下界：
├─ 製作傳送門

升級到獄髓背包：
└─ 鐵砧台：獄髓模板 + 進階背包 + 獄髓錠 → 🟣 獄髓背包 (81格)
```

## 💡 設計理念

### 平衡性考量
1. **基礎背包**：簡單獲取，適合新手
2. **標準背包**：需要鐵器時代進度
3. **進階背包**：需要鑽石裝備階段
4. **獄髓背包**：需要下界探索，遊戲後期

### 升級機制
- **工作台**：基礎製作（符合Minecraft傳統）
- **鐵砧台**：進階升級（符合Minecraft升級系統）

### 材料選擇
- **箱子**：代表儲存，貫穿整個升級鏈
- **鐵錠**：工業時代的象徵
- **鑽石**：珍貴資源，後期進度
- **獄髓**：終極材料，防火特性
