# 背包系統實現完成

## 功能說明

已成功添加背包系統到 DeadRecall 模組，包含以下功能：

### 物品特性
- **物品名稱**: 背包 (Backpack)
- **材質**: 使用 Minecraft 原版的袋子圖片 (`minecraft:item/bundle`)
- **最大堆疊**: 1個（不可堆疊）
- **獲取方式**: 在創造模式物品欄的「工具」分類中

### 背包功能
- **容量**: 27個物品槽（3行×9列，類似箱子）
- **使用方式**: 右鍵點擊背包物品即可開啟UI介面
- **儲存機制**: 物品使用 Data Components 系統儲存，數據會自動保存
- **安全機制**: 防止背包套娃（背包內不能再放背包）

### UI介面
- 使用原版箱子材質的GUI
- 上方3行為背包儲存空間
- 下方為玩家背包
- 支援Shift+左鍵快速移動物品
- 支援滑鼠拖拽操作

## 技術實現

### 新增的檔案

#### Java類別檔案
1. `ModItems.java` - 物品註冊管理
2. `BackpackItem.java` - 背包物品類
3. `BackpackInventory.java` - 背包庫存管理（使用 Data Components API）
4. `BackpackScreenHandler.java` - 背包介面處理器（伺服器端）
5. `BackpackScreen.java` - 背包UI繪製（客戶端）

#### 資源檔案
1. `assets/deadrecall/models/item/backpack.json` - 物品模型定義
2. `assets/deadrecall/lang/en_us.json` - 英文翻譯
3. `assets/deadrecall/lang/zh_tw.json` - 繁體中文翻譯
4. `assets/deadrecall/lang/zh_cn.json` - 簡體中文翻譯

### 修改的檔案
- `Deadrecall.java` - 添加物品註冊初始化
- `DeadrecallClient.java` - 添加客戶端介面註冊

## 如何測試

1. **構建模組**
   ```powershell
   .\gradlew build
   ```

2. **啟動遊戲**
   ```powershell
   .\gradlew runClient
   ```

3. **測試步驟**
   - 進入遊戲創造模式
   - 打開物品欄（E鍵）
   - 在「工具」分類中找到背包物品
   - 將背包拖到快捷欄
   - 右鍵點擊背包開啟介面
   - 測試放入和取出物品
   - 測試Shift+左鍵快速移動
   - 關閉介面後再次開啟，驗證物品是否保存
