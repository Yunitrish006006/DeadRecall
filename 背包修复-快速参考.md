## 🚀 立即测试

### 1. 构建项目

**使用脚本（推荐）**:
```powershell
# 双击运行任一脚本
build-fixed-backpack.bat
build-fixed-backpack.ps1
```

**使用命令行**:
```powershell
.\gradlew build
```

**使用 IDE**:
```
右键项目 → Build Module
或按 Ctrl+F9
```

### 2. 启动游戏
```powershell
.\gradlew runClient
```

### 3. 测试背包
```
1. 创造模式获取背包（工具分类）
2. 右键打开背包
3. 放入10个泥土
4. 关闭背包（ESC）
5. 再次右键打开
6. ✅ 验证10个泥土还在
```
