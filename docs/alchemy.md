# 煉金、材料與配方

DeadRecall 的生存煉金系統將豬糞、木灰、硝石、缽與煉藥鍋串成可取得火藥及飲品的流程。

## 主要材料

| 內容 | 來源或用途 |
| --- | --- |
| 豬糞 | 豬吃草後可能生成帶豬糞的地面方塊；也可作為投擲物與煉金材料 |
| 木灰 | 將乾草捆放入熔爐燒製取得 |
| 硝石 | 在營火加熱的滿水煉藥鍋中，以指定材料完成煉製 |
| 缽 | 用於承裝材料；可對硫磺方塊互動取得帶硫磺的缽 |
| 火藥 | 使用煉金配方合成 |

## 煉藥鍋配方

資料驅動配方位於：

`src/main/resources/data/deadrecall/deadrecall/cauldron_recipes/`

目前包含：

- `saltpeter.json`：木灰、蘑菇與豬糞煉製硝石。
- `hot_cocoa.json`：牛奶、裝可可粉的缽與糖煉製熱可可。
- `cherry_brew.json`：糖、櫻花樹葉、發光莓與甜莓煉製櫻花釀。

## 相關進度

- 取得豬糞可開啟 DeadRecall 進度頁。
- 用豬糞命中生物或玩家有各自對應進度。
- 合成缽會完成煉金器皿相關進度。

配方的精確材料、數量、投料順序與等待條件，以資料檔及遊戲內提示為準。
