## 快速說明

一般背包右鍵開啟，內容使用原版 Container Data Component 儲存。背包不可堆疊，並支援 Shift+點擊快速移動。

開啟背包期間，作為容器本體的那個 ItemStack 會被鎖定：不能 Shift+左鍵移動、拿起、丟棄，或用快捷列數字鍵／副手鍵交換。其他物品仍可正常 Shift+點擊進出背包。

死亡背包會在玩家死亡後收集新產生的一般掉落物，具備永久保存、紅色光柱與虛空保護。
