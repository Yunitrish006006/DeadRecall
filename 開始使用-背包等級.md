## 📋 合成配方速查

### 🟤 基礎背包（9格，1排）
```
[皮革] [皮革] [皮革]
[皮革] [箱子] [皮革]  ← 箱子外圍皮革
[皮革] [皮革] [皮革]
```
需要：**1箱子 + 8皮革**

### 🟦 標準背包（27格，3排）
```
[皮革] [皮革] [皮革]
[皮革] [基礎] [皮革]  ← 基礎背包外圍皮革
[皮革] [皮革] [皮革]
```
需要：**1基礎背包 + 8皮革**

### ⚫ 進階背包（54格，6排）
```
[鐵錠] [鐵錠] [鐵錠]
[鐵錠] [標準] [鐵錠]  ← 標準背包外圍鐵錠
[鐵錠] [鐵錠] [鐵錠]
```
需要：**1標準背包 + 8鐵錠**

## 🎮 使用方式

1. **獲取背包**（三選一）
   - 創造模式：物品欄 → 工具分類
   - 生存模式：按配方合成
   - 指令：`/give @s deadrecall:backpack_basic`

2. **打開背包**
   - 手持背包
   - 右鍵點擊

3. **放入物品**
   - 直接拖拽
   - 或 Shift+左鍵快速移動

4. **自動保存**
   - 關閉界面自動保存
   - 重啟遊戲物品還在

## 💡 升級建議

生存模式推薦路徑：
1. 先殺動物收集8個皮革
2. 製作箱子（8個木板）
3. 合成基礎背包開始用

5. 挖礦後用鐵錠升級到進階
