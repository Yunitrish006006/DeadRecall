## 💡 使用建議

### 生存模式進度
1. 收集材料製作基礎背包（容易）
2. 使用一段時間後升級標準背包（中期）
3. 鐵礦充足後製作進階背包（後期）

### 創造模式
直接使用進階背包即可

---

## 🎊 特色功能

1. **漸進式升級** - 從小到大，循序漸進
2. **動態 UI** - 自動適應背包大小
3. **完整保存** - 使用 Data Components API
4. **防止套娃** - 任何背包都不能放入背包
5. **多語言** - 支援英文、繁中、簡中
