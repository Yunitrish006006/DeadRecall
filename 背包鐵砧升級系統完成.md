# 背包鐵砧升級系統 - 實現完成

## ✅ 新功能：鐵砧台升級系統

### 升級方式改變
- **基礎背包**: 工作台合成（箱子 + 8皮革）
- **其他背包**: 鐵砧台升級（模板 + 背包 + 材料）

---

## 🎒 四種背包等級

### 🟤 基礎背包 (Basic Backpack)
- **容量**: 1排 9格
- **獲取**: 工作台合成
- **配方**:
  ```
  [皮革] [皮革] [皮革]
  [皮革] [箱子] [皮革]
  [皮革] [皮革] [皮革]
  ```

### 🟦 標準背包 (Standard Backpack)
- **容量**: 3排 27格
- **獲取**: 鐵砧台升級
- **配方**: 箱子 + 基礎背包 + 鐵錠

### ⚫ 進階背包 (Advanced Backpack)
- **容量**: 6排 54格
- **獲取**: 鐵砧台升級
- **配方**: 箱子 + 標準背包 + 鑽石

### 🟣 獄髓背包 (Netherite Backpack)
- **容量**: 9排 81格
- **獲取**: 鐵砧台升級
- **配方**: 獄髓模板 + 進階背包 + 獄隨錠
- **特性**: 防火


## 🎮 遊戲內使用

### 製作基礎背包
1. 收集 1個箱子 + 8個皮革
2. 在工作台按配方合成
3. 獲得基礎背包（9格）

### 升級到標準背包
1. 準備 1個箱子 + 1個基礎背包 + 1個鐵錠
2. 打開鐵砧台
3. 放入材料升級
4. 獲得標準背包（27格）

### 升級到進階背包
1. 準備 1個箱子 + 1個標準背包 + 1個鑽石
2. 鐵砧台升級
3. 獲得進階背包（54格）

### 升級到獄髓背包
1. 準備 1個獄髓升級模板 + 1個進階背包 + 1個獄隨錠
2. 鐵砧台升級
3. 獲得獄髓背包（81格，防火）

## 💡 設計理念

### 平衡性考慮
1. **基礎背包**: 簡單獲取，適合早期遊戲
2. **標準背包**: 需要鐵錠，中期進度
3. **進階背包**: 需要鑽石，後期進度
4. **獄髓背包**: 需要下界資源，終極進度

### 升級機制
- 工作台：基礎製作
- 鐵砧台：進階升級
- 符合Minecraft傳統
